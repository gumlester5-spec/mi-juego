# Colección de Juegos Multijugador

Esta es una aplicación web de minijuegos multijugador en tiempo real. Puedes crear una sala, compartir el enlace con un amigo y jugar juntos al instante.

## 🎮 Juegos Incluidos

### 1. Tres en Raya (Tic Tac Toe)
El clásico juego de alinear tres.
**Cómo jugar:**
- Los jugadores se turnan para colocar su símbolo (X o O) en una cuadrícula de 3x3.
- El primer jugador en conseguir 3 símbolos en línea horizontal, vertical o diagonal gana.

### 2. Verdugo (Hangman)
Un juego de adivinar la palabra secreta.
**Cómo jugar:**
- El **Creador de la sala** escribe una palabra secreta (hasta 15 letras).
- El **Invitado** intenta adivinarla letra por letra.
- Si el invitado falla 6 veces, el muñequito se completa y pierde.
- ¡Al terminar, se invierten los roles!

### 3. Puntos y Cajas (Dots and Boxes)
Un juego táctico de cerrar cuadrados.
**Cómo jugar:**
- Los jugadores se turnan para dibujar una línea horizontal o vertical entre dos puntos adyacentes.
- Si una línea completa el cuarto lado de una caja (cuadrado), el jugador se anota un punto y ¡gana un turno extra!
- El juego termina cuando no quedan líneas por trazar. Gana quien haya cerrado más cajas.

### 4. Damas (Checkers)
El clásico juego de mesa de estrategia y capturas.
**Cómo jugar:**
- Las piezas se mueven en diagonal hacia adelante, un cuadro a la vez.
- Si una pieza rival está frente a ti y el cuadro detrás de ella está libre, **debes saltarla y capturarla**. (Las capturas son obligatorias).
- Si al llegar al lado opuesto del tablero coronas, tu pieza se convierte en "Dama" y podrá moverse y comer hacia atrás.
- Gana quien deje sin piezas o sin movimientos válidos a su rival.

### 5. Ludo (Parchís)
Carrera de fichas con dados y mucha suerte.
**Cómo jugar:**
- Los jugadores tiran el dado por turnos para avanzar sus fichas.
- Necesitas un **6 para salir** de casa.
- Si caes en una casilla ocupada por el rival (que no sea una estrella segura), **lo envías de vuelta a casa**.
- Sacar un 6, comer a un oponente o llegar a la meta otorga un **turno extra**.
- El primero en llevar sus 4 fichas al centro, gana.

## 🚀 Cómo empezar
1. Entra a la aplicación.
2. Ingresa tu nombre o apodo.
3. Elige un juego y crea una sala.
4. Comparte el enlace que se genera con tu oponente para que se una y ¡que empiece la partida!
